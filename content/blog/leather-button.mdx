# Making a Leather Button

I saw this really cool leather button design on Twitter the other day and thought it'd be fun to try recreating it. 
There's something satisfying about skeuomorphic designâ€”making digital things look like real objects. 

So here's how I built it with React, Tailwind, and Motion.

## What you'll need

- React project (I'm using Next.js but whatever works)
- Tailwind CSS
- Motion: `npm install motion`

## Starting point

First, just get a basic button working. I used `motion.button` so we can add that satisfying press animation later.

Make a file called `LeatherButton.jsx`:

```jsx
"use client";
import { motion } from "motion/react";

const LeatherButton = () => {
  return (
    <div className="flex items-center justify-center font-sans">
      <motion.button
        className="relative px-12 py-6 rounded-full overflow-hidden focus:outline-none"
        whileTap={{ scale: 0.98 }}
        transition={{ duration: 0.15, ease: "easeOut" }}
      >
        <span className="relative z-10 block text-3xl font-bold tracking-normal">
          Leather Button
        </span>
      </motion.button>
    </div>
  );
};

export default LeatherButton;
```

But overall the button still looks Pretty boring.

## Making it look like leather

Real leather isn't one flat color. It has lighter and darker areas. So instead of a solid background, I used a gradient that goes from lighter brown to darker brown.

```jsx
<motion.button
  className="relative px-12 py-6 rounded-full overflow-hidden focus:outline-none"
  style={{
    background: "linear-gradient(135deg, #6d5447 0%, #5d4037 30%, #4a342a 70%, #3e2723 100%)",
  }}
  // ... rest of props
>
```

  <LeatherButtonStage stage={2} />
</div>


Better already.

## Adding depth with shadows

This is the important part. Real objects cast shadows and catch light. I stacked a bunch of box shadows to make the button look 3D. Some inset to make it look curved, some regular to lift it off the page.

```jsx
style={{
  background: "linear-gradient(135deg, #6d5447 0%, #5d4037 30%, #4a342a 70%, #3e2723 100%)",
  boxShadow: `
    inset 0 1px 2px rgba(255, 255, 255, 0.15),
    inset 0 -3px 6px rgba(0, 0, 0, 0.5),
    0 6px 12px rgba(0, 0, 0, 0.6),
    0 1px 3px rgba(0, 0, 0, 0.4)
  `,
}}
```

The first inset shadow adds a tiny highlight at the top. The second darkens the bottom to make it look rounded. The last two lift it off the background.


<div className="max-w-max m-auto mt-4 mb-6 py-2 px-4 flex flex-col gap-8 shadow-sm outline-1 outline-zinc-300 dark:outline-zinc-700">

</div>


When you press it, the shadow should get smaller (like it's moving closer to the surface):

```jsx
whileTap={{
  scale: 0.98,
  boxShadow: `
    inset 0 1px 2px rgba(255, 255, 255, 0.15),
    inset 0 -3px 6px rgba(0, 0, 0, 0.5),
    0 3px 6px rgba(0, 0, 0, 0.6),
    0 1px 2px rgba(0, 0, 0, 0.4)
  `,
}}
```

## Adding texture

It still looked too smooth more like plastic than leather. Real leather has grain and imperfections.

I used an SVG noise filter instead of loading an image. Add this inside your button:

```jsx
{/* Noise Texture */}
<div
  className="absolute inset-0 opacity-25 pointer-events-none rounded-full"
  style={{
    backgroundImage: `url(
                "data:image/svg+xml,%3Csvg viewBox='0 0 200 200'
                xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter
                id='noiseFilter'%3E%3CfeTurbulence
                type='fractalNoise'
                baseFrequency='1.5'
                numOctaves='3'
                stitchTiles='stitch'/%3E%3C/filter%3E%3Crect
                width='100%25'
                height='100%25'
                filter='url(%23noiseFilter)'/%3E%3C/svg%3E")`,
    backgroundSize: "150px 150px",
  }}
/>

{/* Sheen */}
<div
  className="absolute inset-0 opacity-20 pointer-events-none rounded-full"
  style={{
    background: "linear-gradient(135deg, rgba(255, 255, 255, 0.2) 0%, transparent 50%, rgba(0, 0, 0, 0.3) 100%)",
  }}
/>
```


<div className="max-w-max m-auto mt-4 mb-6 py-2 px-4 flex flex-col gap-8 shadow-sm outline-1 outline-zinc-300 dark:outline-zinc-700">
  <LeatherButtonStage stage={4} />
</div>


The noise breaks up the solid color and the sheen overlay adds a subtle highlight.

## Gold foil text

Plain white text looked okay but I wanted it to look like gold foil stamped into leather. You can do this by making the text transparent and filling it with a gradient.

```jsx
<span
  className="relative z-10 block text-3xl tracking-normal"
  style={{
    backgroundImage: "linear-gradient(180deg, #f5e6d3 0%, #e8d4b0 30%, #d4af87 60%, #c19a6b 100%)",
    WebkitBackgroundClip: "text",
    WebkitTextFillColor: "transparent",
    backgroundClip: "text",
    fontWeight: "700",
  }}
>
  Leather Button
</span>
```


<div className="max-w-max m-auto mt-4 mb-6 py-2 px-4 flex flex-col gap-8 shadow-sm outline-1 outline-zinc-300 dark:outline-zinc-700">
  <LeatherButtonStage stage={5} />
</div>


## Stitching detail

Last touch will be adding a stitching line to make it look sewn. Just a dashed border slightly inside the edges.

## Full code

Here's everything together:

```jsx
"use client";
import { motion } from "motion/react";

const LeatherButton = () => {
  return (
    <div className="flex items-center justify-center font-sans">
      <motion.button
        className="relative px-12 py-6 rounded-full overflow-hidden focus:outline-none"
        style={{
          background: "linear-gradient(135deg, #6d5447 0%, #5d4037 30%, #4a342a 70%, #3e2723 100%)",
          boxShadow: `
            inset 0 1px 2px rgba(255, 255, 255, 0.15),
            inset 0 -3px 6px rgba(0, 0, 0, 0.5),
            0 6px 12px rgba(0, 0, 0, 0.6),
            0 1px 3px rgba(0, 0, 0, 0.4)
          `,
        }}
        whileTap={{
          scale: 0.98,
          boxShadow: `
            inset 0 1px 2px rgba(255, 255, 255, 0.15),
            inset 0 -3px 6px rgba(0, 0, 0, 0.5),
            0 3px 6px rgba(0, 0, 0, 0.6),
            0 1px 2px rgba(0, 0, 0, 0.4)
          `,
        }}
        transition={{ duration: 0.15, ease: "easeOut" }}
      >
        {/* Noise Texture */}
        <div
          className="absolute inset-0 opacity-25 pointer-events-none rounded-full"
          style={{
            backgroundImage: `url(
                "data:image/svg+xml,%3Csvg viewBox='0 0 200 200'
                xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter
                id='noiseFilter'%3E%3CfeTurbulence
                type='fractalNoise'
                baseFrequency='1.5'
                numOctaves='3'
                stitchTiles='stitch'/%3E%3C/filter%3E%3Crect
                width='100%25'
                height='100%25'
                filter='url(%23noiseFilter)'/%3E%3C/svg%3E")`,
            backgroundSize: "150px 150px",
          }}
        />

        {/* Sheen */}
        <div
          className="absolute inset-0 opacity-20 pointer-events-none rounded-full"
          style={{
            background: "linear-gradient(135deg, rgba(255, 255, 255, 0.2) 0%, transparent 50%, rgba(0, 0, 0, 0.3) 100%)",
          }}
        />

        {/* Stitching */}
        <div
          className="absolute inset-0 rounded-full pointer-events-none"
          style={{
            top: "4px",
            left: "4px",
            right: "4px",
            bottom: "4px",
            border: "1.5px dashed rgba(139, 107, 83, 0.8)",
            borderRadius: "9999px",
            boxShadow: "inset 0 1px 1px rgba(0,0,0,0.2)"
          }}
        />

        {/* Text */}
        <span
          className="relative z-10 block text-3xl tracking-normal"
          style={{
            backgroundImage: "linear-gradient(180deg, #f5e6d3 0%, #e8d4b0 30%, #d4af87 60%, #c19a6b 100%)",
            WebkitBackgroundClip: "text",
            WebkitTextFillColor: "transparent",
            backgroundClip: "text",
            fontWeight: "700",
          }}
        >
          Leather Button
        </span>
      </motion.button>
    </div>
  );
};

export default LeatherButton;
```

That's it. The trick is layering i.e. gradients for color variation, shadows for depth, noise for texture, and small details like stitching.
